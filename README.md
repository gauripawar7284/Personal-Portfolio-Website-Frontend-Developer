Getting Started
Prerequisites
You only need a modern web browser to run this project.

Installation
Clone the repository:
git clone https://github.com/your-username/personal-portfolio-website.git

Navigate to the project folder:
cd personal-portfolio-website
Open index.html in your browser.

Usage
Update your personal details in index.html
Replace images inside the assets folder
Customize colors, fonts, and layout in style.css
Add or modify interactions in script.js
